# VisionARies - AR Heritage Experience
A TikTok AR Effect developed to promote Malaysian traditional games while collaborating with BADAN WARISAN MALAYSIA.

Course: WIX2001: THINKING AND COMMUNICATION SKILLS
Semester: SEMESTER 3

## Platform
- TikTok Effect House

## Concept
Using augmented reality to introduce traditional Malaysian games to younger audiences.

## Heritage Elements
- Sepak takraw
- Wau
- Gasing
